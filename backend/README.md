# Shraddha Traders — Backend

This folder will contain the Node.js + Express backend.

## Coming Soon
- REST API for products, customers, suppliers, sales
- MySQL database with proper schema
- JWT authentication
- Firebase integration option

## For Now
The frontend runs completely without this backend.
All data is stored in browser memory using React state.

## To Start Backend Later
```bash
cd backend
npm init -y
npm install express mysql2 bcryptjs jsonwebtoken cors dotenv
```
